# E-Commerce Sales Analysis Dashboard

## Project Overview
This project analyzes e-commerce sales data to evaluate business performance, identify trends, and support executive decision-making using Power BI.

## Key Features
- Sales trend analysis (Monthly growth)
- Regional performance analysis
- Product category insights
- Customer segmentation
- KPI tracking (Sales, Orders, AOV)

## Tools Used
- Power BI
- DAX
- Data Modeling

## Key Insights
- Business shows positive growth trends
- Certain regions dominate revenue
- High-value customers contribute most revenue
- Seasonal patterns observed

## Limitations
Profit margin could not be calculated due to lack of cost data.
